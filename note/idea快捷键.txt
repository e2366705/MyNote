idea几个比较常用的快捷键:
--------------------------------------------------
ctrl + alt + T 
psvm + Tab
alt + insert 生成 get + set + 构造方法...
ctrl + O  重写父类doget + dopost方法!!!
ctrl + alt + S 打开idea的设置大全 settings
重命名项目/文件  shift + F6






path  是访问应用的路径，例如我上面写的是/SSM，则访问路径是     http://localhost/SSM
port 是 Tomcat 的端口号　　
uriEncoding  URL按UTF-8进行编码，这样就解决了中文参数乱码
Server 指定tomcat名称

  <build>
    <!-- 配置了很多插件 -->
    <plugins>

      <!-- 编译插件 -->
      <plugin>
        <groupId>org.apache.tomcat.maven</groupId>
        <artifactId>tomcat7-maven-plugin</artifactId>
        <version>2.2</version>
        <configuration>
          <port>8080</port>
          <path>/SSM</path>
          <uriEncoding>UTF-8</uriEncoding>
          <server>tomcat7</server>
        </configuration>
      </plugin>

    </plugins>
  </build>


  tomcat 常用命令:
        命令               描述
    tomcat7:deploy    部署一个web war包
    tomcat7:reload    重新加载web war包
    tomcat7:start

    启动tomcat
    tomcat7:stop

    停止tomcat
    tomcat7:undeploy

    停止一个war包
    tomcat7:run    启动嵌入式tomcat ，并运行当前项目



idea 下载安装设置:
    1 设置字体: 
        两种字体: 1 代码区编辑器字体: 在 settings 搜索: font
                 2 软件的界面字体 打开settings -> 搜索: appearance -> 右边设置软件界面字体
    2 设置快捷键:
        打开 settings -> 搜索: keymap 设置快捷键
