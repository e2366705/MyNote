# vs code设置注视颜色是粉色(#F69AB6):
## 点击 VS code 左下角的齿轮(设置) -> 搜索 "settings.json":
```  
{
    "window.zoomLevel": 3,
    "workbench.colorTheme": "Monokai",
    "editor.tokenColorCustomizations": {
        "comments": "#F69AB6", // 注释 = 粉色
      }
}


```
